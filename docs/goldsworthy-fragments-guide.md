# Goldsworthy Fragments Generator
*Adaptive energy signatures that respond to article content*

A system that analyzes your writing and generates broken, beautifully rearranged energy fragments - like Andy Goldsworthy taking your ideas, smashing them thoughtfully, and arranging the pieces into natural beauty.

---

## Philosophy

> "Take something whole and sophisticated, smash it thoughtfully, then arrange the pieces with natural intuition. The fragments should hold memory of the original energy while finding new harmony in their brokenness."

**Core Principles:**
- **Beautiful imperfection** - Asymmetrical, organic placement
- **Temporal quality** - Gentle movements suggest natural change  
- **Found harmony** - Fragments relate but don't align perfectly
- **Essential simplicity** - Complex energy reduced to pure elements
- **Natural breathing** - All pieces move with organic rhythms

## Content Analysis System

### **Stage 1: Article Essence Extraction**

**Emotional Tone Analysis:**
- **Contemplative** → More circular, inward-flowing fragments
- **Energetic** → Angular, outward-radiating fragments  
- **Technical** → Linear, structured fragment patterns
- **Personal** → Organic, irregular shapes
- **Creative** → Asymmetrical, expressive arrangements

**Content Complexity Mapping:**
- **Simple concept** → 2-3 fragments, clear spacing
- **Layered ideas** → 4-5 fragments, overlapping elements
- **Complex systems** → 6+ fragments, intricate relationships
- **Abstract concepts** → Ethereal, fading fragments
- **Concrete processes** → Sharp, defined fragment edges

**Thematic Keywords → Fragment Selection:**
- **"growth", "development", "journey"** → Broken Circle Arc (incomplete cycles)
- **"inspiration", "creativity", "spark"** → Scattered Radial Pieces (dispersed energy)
- **"process", "method", "steps"** → Broken Line Energy (directional flow)
- **"natural", "organic", "adaptive"** → Irregular Organic Shapes
- **"core", "essence", "truth"** → Always include Core Essence (universal)

### **Stage 2: Fragment Generation Logic**

**Article Length Influence:**
```
Short (< 500 words):    3 fragments, simple arrangement
Medium (500-1500):      4-5 fragments, balanced complexity  
Long (1500+):           5-7 fragments, rich interconnections
```

**Section Count Influence:**
```
1-2 sections:           Linear arrangement
3-4 sections:           Triangular composition
5+ sections:            Organic, scattered arrangement
```

**Paragraph Rhythm Analysis:**
```
Short, punchy paragraphs:    Sharp, angular fragments
Long, flowing paragraphs:    Smooth, curved fragments
Mixed rhythm:                Combination of both styles
```

## Adaptive Fragment Types

### **1. Broken Circle Arc**
- **Purpose**: Represents incomplete cycles, unfinished thoughts, potential
- **Form**: Circle with missing segments (top, right, bottom, or left)
- **Best for**: Reflective articles, personal growth, learning journeys

```css
border: 2px solid rgba(199, 190, 181, 0.6);
border-top: none;        /* Remove top segment */
border-right: none;      /* Remove right segment */
border-radius: 50%;
```

### **2. Scattered Radial Piece**
- **Purpose**: Represents dispersed energy, inspiration, fleeting moments
- **Form**: Soft radial gradient that drifts naturally
- **Best for**: Creative process articles, inspiration pieces, scattered thoughts

```css
background: radial-gradient(circle,
  rgba(199, 190, 181, 0.5) 0%,
  rgba(199, 190, 181, 0.3) 40%,
  transparent 70%
);
```

### **3. Broken Line Energy**
- **Purpose**: Represents interrupted flow, partial connections, traces
- **Form**: Linear gradient that fades to nothing
- **Best for**: Technical articles, process documentation, work-in-progress

```css
background: linear-gradient(90deg,
  rgba(199, 190, 181, 0.7) 0%,
  rgba(199, 190, 181, 0.4) 60%,
  transparent 100%
);
```

### **4. Organic Irregular Shape**
- **Purpose**: Represents natural evolution, adaptive thinking, organic growth
- **Form**: Asymmetrical blob with changing border-radius
- **Best for**: Nature-inspired articles, adaptive processes, organic development

```css
border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%;
/* Animate between different organic shapes */
```

### **5. Core Essence**
- **Purpose**: Represents the essential truth, the heart of the matter
- **Form**: Small, bright, pulsing circle
- **Best for**: All articles - this is the central creative spark

```css
background: rgba(199, 190, 181, 0.8);
border-radius: 50%;
filter: drop-shadow(0 0 6px rgba(199, 190, 181, 0.4));
```

## Composition Guidelines

### **Spacing**
- **Container**: 80px × 60px (standard proportions)
- **Fragment spacing**: Minimum 8px between elements
- **Overlap**: Allow slight overlap for natural feel

### **Hierarchy**
1. **Core Essence** - Always brightest, most prominent
2. **Primary Fragment** - Main conceptual element
3. **Secondary Fragments** - Supporting elements
4. **Ambient Field** - Subtle background glow

### **Balance**
- **Asymmetrical** - Never center or align perfectly
- **Weighted** - Heavier elements toward bottom-left
- **Breathing room** - Don't fill entire container

### **Sentiment Analysis → Fragment Character**

**Positive/Optimistic Content:**
- Fragments point upward and outward
- Brighter opacity values
- Expanding, growing animations
- Warmer color shifts

**Reflective/Introspective Content:**
- Fragments curve inward
- Softer, contemplative movements  
- Circular and arc-based shapes
- Gentle, breathing rhythms

**Analytical/Critical Content:**
- Sharp, angular fragments
- Precise, measured animations
- Linear and geometric patterns
- Cooler, more structured arrangements

**Experimental/Creative Content:**
- Irregular, asymmetrical shapes
- Unpredictable animation patterns
- Bold contrasts in opacity
- Playful, expressive movements

### **Keyword Density Mapping**

**High Keyword Density (focused topic):**
- Tighter fragment clustering
- More unified movement patterns
- Stronger central core essence
- Related fragments orbit the core

**Low Keyword Density (broad exploration):**
- Scattered fragment arrangement
- Independent animation rhythms
- Multiple competing elements
- Decentralized composition

**Technical Terminology Frequency:**
```
High tech terms:        Geometric, precise fragments
Medium tech terms:      Mixed organic/geometric
```

- Can be used as section dividers
- Smaller scale (60px × 45px) for inline use
- Reduce animation intensity for readability

### **Responsive Considerations**
- Scale down proportionally on mobile
- Reduce animation on `prefers-reduced-motion`
- Maintain fragment relationships across screen sizes
